# Design Direction

PraktijkOS should feel like a calm operational workspace for Belgian care practices.

## Principles

- Dense enough for daily use, never a marketing dashboard.
- Light navigation, compact cards and quiet status colors.
- Use panels for real work surfaces, not decorative page sections.
- Keep actions close to the object they affect.
- Avoid medical drama, gradients and oversized hero patterns.
- Maintain 8px radius for cards, panels and controls.

## Current UI Direction

- Light sidebar with compact navigation.
- White work panels on a pale neutral background.
- Muted labels and clear operational headings.
- Status badges with soft semantic color.
- Responsive horizontal navigation on smaller screens.
